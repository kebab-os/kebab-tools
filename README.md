## What is kebab-tools?

Kebab-tools is a collection of tools that you can curl from the command line terminal. It has lots of useful tools. It is very useful for developers, who dont want to leave the CLI.


## Get started

1. Start by running this bash command in shell:

```bash
curl https://tools.kebabos.me
```

If that doesnt work, try our secondary domain:

```bash
curl https://kebab-tools.pages.dev
```

2. If you want to view all the tool directories, run `/list` after the curl command.

If this doesnt work, [raise an issue](https://github.com/kebab-os/kebab-tools/issues/new/choose) to help support the project.
